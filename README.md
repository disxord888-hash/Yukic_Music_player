# yukic music player

YouTube, SoundCloud, Vimeo の動画や音声を再生・管理できる高機能なウェブベースのミュージックプレイヤーです。

## 主な特徴
- **マルチプラットフォーム対応**: YouTube, SoundCloud, Vimeo の動画をシームレスに再生。
- **高機能なシーク**: キーボードショートカット（A/S/D/F/H/J/K/L）でフレーム単位や秒単位の精密なシークが可能。
- **キュー管理**: 再生リスト（キュー）の編集、保存（Export）、読み込み（Import）に対応。
- **アラーム機能**: 指定した時間に特定の曲を再生するアラーム設定。
- **PWA対応**: Google Chrome でデスクトップアプリとしてインストール（ダウンロード）可能。

## インストール方法 (Chrome)
1. Google Chrome でサイトを開きます。
2. アドレスバーの右側にある「インストール」アイコン（📥）をクリック、またはメニューから「保存して共有」→「インストール」を選択します。
3. デスクトップやアプリ一覧から直接起動できるようになります。

## ショートカットキー
- **G**: 再生 / 一時停止
- **Z / X / C / V**: 最初の曲 / 前の曲 / 次の曲 / 最後の曲
- **A / S / D / F**: -1/36秒 / -1/72秒 / -5秒 / -2秒
- **H / J / K / L**: +2秒 / +5秒 / +1/72秒 / +1/36秒
- **Q / E / W**: 1曲ループ / 全体ループ / シャッフル
- **Ctrl + F**: キュー内検索
- **Shift + / (?)**: Web検索 (Google)
